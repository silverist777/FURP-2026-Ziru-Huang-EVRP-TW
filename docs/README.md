# Project Notes Index

This directory is the required FURP home for weekly progress, meeting notes,
checkpoints, and handoff material. Documents are grouped below by purpose rather
than moved, so existing links and the original repository constraints remain
intact.

## Current checkpoint

- [`week05_checkpoint.md`](week05_checkpoint.md) — latest project status,
  comparison results, limitations, and next steps.
- [`EVRPTW_FOUR_METHODS_HANDOFF.md`](EVRPTW_FOUR_METHODS_HANDOFF.md) — technical
  handoff for continuing the four-method experiment.
- [`src_data_method_run_tutorials.md`](src_data_method_run_tutorials.md) — long
  form commands for individual methods and datasets.

## Progress history

## Meeting notes

Create one file per meeting in [`meeting_notes/`](meeting_notes/) using
[`meeting_notes/TEMPLATE.md`](meeting_notes/TEMPLATE.md). Keep decisions, action
items, and unresolved questions here rather than in transient chat messages.
